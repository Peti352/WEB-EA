Objektumorientált JavaScript

Kattints a vászonra egy új alakzat hozzáadásához. Az alakzatok gravitáció hatására mozognak és pattannak a falakról. Shift + kattintás: csillag alakzat (öröklődik a körből).

Alakzatok száma: 0

Használt OOJS elemek

  • class Shape – alaposztály (constructor, metódusok)
  • class Circle extends Shape – öröklődés super()-rel
  • class Star extends Circle – további örökítés
  • document.body.appendChild – DOM elem beszúrás
