A projekt célja

A Webprogramozás 1 kurzus előadás beadandó feladatához készült web-alkalmazás. A cél bemutatni ugyanazt a CRUD (Create, Read, Update, Delete) logikát hét különböző technológiai megközelítésben, ezzel demonstrálva a modern web-fejlesztés eszköztárát.

Menüpontok

2 pont

JavaScript

CRUD tiszta JavaScripttel, adatok memóriában tárolt tömbben.

2 pont

React

Ugyanaz a CRUD funkcionalitás React komponensekkel és useState hookkal.

3 pont

SPA

Single Page Application két React mini-alkalmazással: Kalkulátor és Amőba.

4 pont

Fetch API

JavaScript + Fetch API, szerver oldali PHP + MySQL adatbázissal.

4 pont

Axios

React + Axios HTTP kliens, ugyanaz a PHP backend.

3 pont

OOJS

Objektumorientált JavaScript grafikus alkalmazás: mozgó alakzatok Canvas-on.

Tech stack

Front-end: HTML5, CSS3, JavaScript (ES6+), React (Vite build). Back-end: PHP 8 + PDO, MySQL / MariaDB. HTTP: Fetch API, Axios. Deploy: Nethely.hu ingyenes tárhely. Verziókövetés: Git + GitHub.
